Generate a dataset of exactly 200 questions for testing a PII detection 
system called SurrogateShield. Output ONLY a valid JSON array — no 
explanation, no markdown, no backticks, just raw JSON starting with [ 
and ending with ].

---

OUTPUT FORMAT — every entry must be exactly this structure:

{
  "Question": "the full question text here",
  "Answer-Key": {
    "type_label": "exact_pii_value_as_it_appears_in_question"
  }
}

For no-PII questions use: "Answer-Key": {}
For multiple values of the same type use a list: "name": ["John", "Jane"]

---

TYPE LABELS — use exactly these strings:

  "name"           full personal names
  "email"          email addresses
  "phone"          any phone number (US, UK, international)
  "ssn"            Social Security Numbers (XXX-XX-XXXX format)
  "address"        street addresses only (not city/state alone)
  "dob"            dates of birth
  "org"            company or organization names
  "gpe"            specific identifying locations (small towns, 
                   neighborhoods — NOT major cities or US states)
  "credit_card"    credit card numbers (must pass Luhn validation)
  "api_key"        API keys or tokens (sk- prefix, Bearer tokens, etc.)
  "ip_address"     IPv4 addresses
  "zip"            US ZIP codes
  "postcode"       UK postcodes
  "gender"         explicit gender declarations
  "crypto"         cryptocurrency wallet addresses
  "bank_number"    ABA routing numbers (9 digits)
  "driver_license" driver's license numbers
